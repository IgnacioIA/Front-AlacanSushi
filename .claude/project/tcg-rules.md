# Mitos y Leyendas — Contexto TCG

## El juego

**Mitos y Leyendas** es un juego de cartas coleccionables (TCG) creado en Chile.
Los jugadores construyen mazos con cartas de distintas expansiones y se enfrentan en duelos.

## Términos importantes

| Término | Significado |
|---------|-------------|
| **Expansión** | Set de cartas (ej: Leyendas 4.0, Shogun) |
| **Display** | Caja sellada con múltiples sobres |
| **Sobre** | Pack de cartas aleatorias |
| **Mazo Inicial** | Mazo pre-construido para principiantes |
| **Kit de Batalla** | Producto con dos mazos para jugar directo |
| **Lootbox** | Edición especial con cartas foil/premium (Edición PBX) |
| **BuyABox** | Promo exclusiva por comprar un display |
| **Carta Secreta** | Carta ultra-rara de la expansión |
| **Carta Foil** | Carta con acabado metálico/brillante |
| **Ban List** | Lista de cartas prohibidas o limitadas en formatos oficiales |
| **Formato** | Reglas sobre qué expansiones son legales en torneos |

## Expansiones en el sitio

### Leyendas 4.0 (Mitos 4.0)
- Ruta: `/mitos4.0`
- Nombre interno del set: `LPB4` (Leyendas Primer Bloque 4)
- Códigos de carta: `LPB4-001`, `LPB4-006`, etc.

### Shogun
- Ruta: `/Shogun`
- Nombre interno del set: `APB5` (Andes Primer Bloque 5)
- Códigos de carta: `APB5-05`, `APB5-25`, etc.
- Tiene fondo hero diferenciado para mobile.

### Edición PBX 2025 (Lootbox)
- Ruta: `/EdicionPBX`
- Edición especial, no un set regular — enfocada en cartas premium.

### Mazos Iniciales Primer Bloque
- Ruta: `/mazoInicialPB`
- Productos de entrada para nuevos jugadores.
